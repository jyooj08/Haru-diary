## How to Run
1. Node.js 설치
https://nodejs.org/en/download/current/

2. express, ejs, mysql 모듈 설치
```
npm install express
npm install ejs
npm install mysql
```  
3. server.js 실행
```
node server.js
```
4. 브라우저를 통해 localhost:50000에 접속
